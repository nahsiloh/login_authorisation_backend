## Files you need to create

Create a `.env` and ensure that the `.env` file is in .gitignore.

```.env
{
  JWT_SECRET_KEY=<Generate your own>
}
```

## Getting started

To start server

```
npm start
```

To start server in development mode

```
npm run start:dev
```

To run the tests

```
npm test
```

To build the app

```
npm run build
```
